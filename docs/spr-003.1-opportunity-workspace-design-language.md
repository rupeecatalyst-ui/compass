## 2. Responsive desktop and tablet layouts

| Breakpoint | Behaviour |
|------------|-----------|
| **Mobile** | Single column; quick actions 2×3; compass stacks metrics under dial |
| **Tablet (md)** | Customer / LIFE / Documents / Tasks in 2×2 |
| **Desktop (xl)** | Main 2×2 + sticky right-side CHANAKYA Insights (`20rem`) |
| **All** | Dialogue Center remains full-width at bottom |

Workspace shell forces a **dark enterprise surface** (`className="dark"`) so the design language is consistent even if the global theme is light.

---

## 3. Summary of UI improvements

| Section | Change |
|---------|--------|
| **Executive header** | Opportunity No., Product, Loan Amount, Customer, Lender, RM, Stage, Progress as info chips |
| **Compass** | Primary visual centrepiece under header — needle, health, status, success probability, pulse |
| **CHANAKYA** | Dedicated advisory band with fixed portrait; one message at a time |
| **KPI strip** | Compact consistent KPI cards (Pulse, Health, Age, Docs, Tasks, Overdue, Activity, RM) |
| **Quick actions** | Large icon cards (Task, Document, Dialogue, Lender, Timeline, Stage) |
| **Customer / LIFE / Docs / Tasks** | Glass panels; circular progress on Docs & Tasks; LIFE summary fields |
| **Insights** | Right-side advisory list |
| **Dialogue** | Full-width glass timeline with existing filters |
| **Design system** | Shared `workspace-design.tsx` primitives (glass, chips, KPIs, circular progress) |

**Unchanged:** business logic, workflows, engines, APIs, decision automation.
